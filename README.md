# Program Data Mahasiswa 

Program ini dibuat untuk mengelola data nilai mahasiswa menggunakan konsep **Class** dengan bahasa pemrograman **Python**.

## Fitur
- Tambah data mahasiswa
- Tampilkan data mahasiswa
- Ubah data mahasiswa
- Hapus data mahasiswa

## Class yang Digunakan
Class: `DaftarNilaiMahasiswa`  
Method:
- `tambah()`
- `tampilkan()`
- `ubah()`
- `hapus()`
