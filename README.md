# Program Data Nilai Mahasiswa

## Deskripsi
Program ini dibuat untuk mengelola data nilai mahasiswa menggunakan konsep OOP (Object Oriented Programming) dengan class.

## Fitur Program
- Tambah data mahasiswa
- Tampilkan semua data
- Hapus data berdasarkan nama
- Ubah data berdasarkan nama

## Bahasa Pemrograman
- Python

## Cara Menjalankan Program
1. Pastikan Python sudah terinstal
2. Buka terminal atau CMD
3. Jalankan perintah:
   python main.py

## Struktur Class
Class DataMahasiswa memiliki atribut:
- data (list)

Dan method:
- tambah()
- tampilkan()
- hapus(nama)
- ubah(nama)
